# Fresher Hiring Prediction using Machine Learning

## Overview
This project predicts whether a fresher candidate will be selected for hiring using a Random Forest Classifier. It includes data exploration, preprocessing, visualization, model training, and performance evaluation.

## Features
- Data quality checks (missing and duplicate values)
- Exploratory Data Analysis (EDA) with visualizations
- Categorical data encoding using Label Encoding
- Data balancing using SMOTE
- Train-test split for model validation
- Random Forest classification model
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - ROC-AUC Score
  - Confusion Matrix
  - Classification Report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Imbalanced-learn (SMOTE)

## Workflow
1. Load and inspect the dataset.
2. Check for missing and duplicate values.
3. Perform exploratory data analysis.
4. Remove unnecessary columns.
5. Encode categorical features.
6. Split the dataset into training and testing sets.
7. Balance the training data using SMOTE.
8. Train a Random Forest Classifier.
9. Evaluate the model using multiple classification metrics.

## Result
The trained model predicts whether a candidate is likely to be selected based on academic performance, technical skills, internship experience, certifications, and other relevant features.
